# עירוני

דשבורד ערים בישראל המבוסס על נתוני למ"ס בזמן אמת דרך `data.gov.il`.

## מה האתר עושה

- טוען נתוני ערים ישירות מ־API פתוח של הלמ"ס
- מציג חיפוש ובחירת עיר
- מציג KPI לכל עיר
- מציג גרף השוואת אוכלוסייה בין ערים מובילות
- מציג גרפי היסטוריה לעיר נבחרת: אוכלוסייה, שינוי שנתי, ודירוג אוכלוסייה ארצי
- מאפשר השוואת דירוג היסטורי בין שתי ערים על אותו גרף

## מקורות נתונים (רשמיים)

1. מאגר `localities-in-israel` (קובצי יישובים 2018-2023)
2. מאגר `2022` (מפקד האוכלוסין והדיור 2022) - משאב "נתונים נבחרים לפי יישובים ואזורים סטטיסטיים"

האפליקציה ממזגת את המקורות לפי סמל יישוב.

## הרצה

האתר סטטי ודורש שרת מקומי:

```bash
python -m http.server 8000
```

פתיחה בדפדפן:

`http://localhost:8000`

## נפילות רשת / API

אם טעינת ה־API נכשלת, המערכת עוברת אוטומטית לקובץ גיבוי מקומי:

- `data/cities-sample.json`

## קבצים מרכזיים

- `index.html` - מבנה הדף
- `styles.css` - עיצוב
- `app.js` - טעינת API, מיזוג נתונים ולוגיקת הדשבורד
- `data/cities-sample.json` - נתוני גיבוי
